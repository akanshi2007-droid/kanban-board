# Kanban Task Board
A modern **Kanban Task Management Application** built using **React.js**.
This application helps users organize tasks visually using a **drag-and-drop Kanban board** with columns like **To Do, In Progress, and Done**.
Users can create tasks, move them between columns, search tasks, and track progress easily.

## How It Works
1. Users can add tasks using the **Add Task modal**.
2. Tasks appear in the **To Do column**.
3. Tasks can be **dragged and dropped** into:

   * In Progress
   * Done
4. Users can **search tasks** using the search bar.
5. Tasks can be **deleted** when completed.
